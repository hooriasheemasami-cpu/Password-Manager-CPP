# Password Manager (C++)

A simple console-based Password Manager built using C++. The application allows users to store, retrieve, update, and manage login credentials through a menu-driven interface.

This project was developed to strengthen my understanding of C++ programming, object-oriented programming (OOP), file handling, and software design. It also provided practical experience in building a complete console application and managing a software project using Git and GitHub.

> **Note:** This project was created by following a guided C++ tutorial as part of my software development learning journey. The code has been implemented for educational purposes while helping me understand the core concepts behind file-based password management systems.

---

## Features

- Add new credentials
- View saved credentials
- Search for existing credentials
- Update stored passwords
- Delete saved credentials
- Store data locally using text files
- Menu-driven console interface
- Basic input validation

## How It Works

1. Launch the application.
2. Choose an option from the main menu.
3. Enter website, username, and password details.
4. Save credentials to a local text file.
5. Search, update, view, or delete stored credentials whenever needed.

---

## Installation

Clone the repository:

```bash
git clone https://github.com/hooriasheemasami/Password-Manager-CPP.git
```

Navigate to the project directory:

```bash
cd Password-Manager-CPP
```

Compile the program:

```bash
g++ main.cpp -o PasswordManager
```

Run the application.

**Windows**

```bash
PasswordManager.exe
```

**Linux/macOS**

```bash
./PasswordManager
```
